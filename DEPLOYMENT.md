# Deployment Guide

## Local Docker

```bash
cp .env.example .env
docker compose up --build
```

Open:

```text
http://localhost:8000/docs
```

## Cloud Deployment

Deploy this as a Docker app on Render, Railway, Fly.io, ECS, or EKS.

## Safe Automation

Use the approval queue and scheduler. Do not mass-post or duplicate-spam content.
